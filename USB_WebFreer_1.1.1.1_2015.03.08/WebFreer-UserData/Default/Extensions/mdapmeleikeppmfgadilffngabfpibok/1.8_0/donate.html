A message from developers
Due to new Google Chrome policies, we have decided to limit the functionalities on Facebook Messenger, since otherwise it would soon be removed from the Chrome Web Store. We believe that Facebook Messenger provides an additional tool for any Facebook user, since it is focused on private conversations, disregarding (often less important) other kind of notifications.
Therefore, we have decided to make the extension respect the new policies. By doing that, we have removed the ability to surf Facebook website. Since we know that this function is fundamental for some of you, we have decided to create a new extension focused on that. The new extension is called Facebook Mobile View and it's already available for free on the Google Chrome Web Store.

Thanks for your understanding
